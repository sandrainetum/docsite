  • Module guides

    This section provides all module guides organized by module.

    • Cleva Analytics

      This module includes several data marts generated based on the business of Life, Non-life and Accounting areas. It allows extracting, from massive data volumes, information that is relevant for a quick decision-making process.

    • Cleva Frontend factory

      This module provides features to define the UX/UI of all Cleva software. It allows insurers to define menus, pages and sections, and to integrate different Cleva modules within the same user interface.

    • Cleva Outputs

      This module supports integration between Cleva software and third-party applications regarding output management, allowing document generation, sending and archiving.

      • General information

        This chapter introduces overview information about Cleva Outputs, such as its architecture, roles and other resources.

      • Flow
        • Flow configurator

          This feature provides a console that allows configuring the flow steps and actions for each event that might generate an output.

        • Flow traceability

          This feature allows viewing all the steps made during the flow life cycle.

        • Monitoring console

          This feature provides a console that allows viewing the status of any output’s flow instance.

      • Enrichers
      • Document generation

        This chapter refers to the phase when the document is generated, based on data merged layout. This step may also be a batch operation, in which case no return is expected.

      • Document archiving

        This chapter refers to the document management, where it is possbile, for example, to archive a document or search for stored documents.

      • Configurations
    • Cleva Product factory

      This module includes product engineering features, which allow the insurance company to configure all the components the system needs to manage a product, including tariffs, data validation rules, risk acceptance, calculation of taxes and commissions, claim context, indemnity calculation rules and the behaviour expected in a specific distribution channel.

    • Cleva Technical accounting

      Cleva Technical accounting (TAC) is a solution designed to support the integration of the insurance company’s core system with its accounting system.

    • Cleva Workflow

      This module includes the workflow engine that orchestrates all business processes and corresponding insurance activities, from underwriting to claim management. It allows implementing activity automation according to the business rules defined by the insurance company, and the activities processed manually are available to users via the task manager.

Flow configurator

This feature provides a console that allows configuring the flow steps and actions for each event that might generate an output.

The Cleva Outputs’ flow allows configuring the following items:

  • event definition;
  • available actions;
  • enrichers;
  • documents;
  • notifications;
  • dispatch modes;
  • layouts.
