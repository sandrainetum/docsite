    • Cleva Outputs

      This module supports integration between Cleva software and third-party applications regarding output management, allowing document generation, sending and archiving.

    • Cleva Party management

      This module manages the party management functional area, common to Life and Non-life. In particular, this module is a software layer that integrates with Cleva Common area v6, Cleva Common area v7 and Cleva Life v7.

    • Cleva Process factory

      This module allows insurers to define their own business processes. Based on the concept of product library, which has several components integrated with the Cleva software, the insurer is provided with accelerators to address business processes such as underwriting and claims.

    • Cleva Product factory

      This module includes product engineering features, which allow the insurance company to configure all the components the system needs to manage a product, including tariffs, data validation rules, risk acceptance, calculation of taxes and commissions, claim context, indemnity calculation rules and the behaviour expected in a specific distribution channel.

    • Cleva Security manager

      This framework supports features associated with security and roles. It allows insurers to create profiles and to define which features shall apply to them.

    • Cleva Technical accounting

      This module supports the integration of the insurance companyâs technical data with the corresponding accounting system. This means that, with this module, it is possible to process the documents/events produced by the different operations and automatically integrate them in the accounting system.

    • Cleva Workflow

      This module includes the workflow engine that orchestrates all business processes and corresponding insurance activities, from underwriting to claim management. It allows implementing activity automation according to the business rules defined by the insurance company, and the activities processed manually are available to users via the task manager.

View accounting transactions

This feature provides an area to view the accounting transactions of the receipts in the receipt details interface.

Thus, in the receipt details interface, there is a new area available where you can view the accounting transactions associated with the corresponding receipt.
